Introduction

HTML est basé sur le langage Hyper Text MarkUp, il decrit la structure de la page Web.
Il consiste en une série d'éléments qui vont dire au navigateur comment les afficher.
ces éléménts sont representé par des tags.
ces différent tags peuvent contenir des informations comme "heading", "paragraph",table" etc... .
Le navigateur n'affiche pas ces tags mais s'en sert pour le rendu du contenu de la page.

Les navigateurs WEB

Le but des navigateurs (Chrome, Edge, Firefox...) est de lire les documents HTML et de les afficher.
Le navigateur n'affiche pas les tags HTML, mais s'en sert pour determiner comment afficher le doucment.

les éditeurs HTML

On peut créer et modifier des pages HTML en utilisant des éditeurs professionnel HTML.
Néanmoins pour pour apprendre le HTML, un simple éditeur de texte comme Notepad est recommandé.

Liste des principaux tags

  • <html> Racine du doc Html
  • <body> Corps du document
  • <head> Conteneur avec les éléments d'en-tête (titre, script, style...)
  • <h1>to <h6>defini les en-tête de plus gros au plus petit
  • <hr> défini de changement de conteneur
  • <p> défini un paragraphe
  • <br> insère un saut de ligne
  • <pre> defini un texte prédéfini
